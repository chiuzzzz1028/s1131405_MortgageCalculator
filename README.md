s1131405_MortgageCalculator
# 專案簡介
本專案為視窗程式設計課程(二)作業，旨在提供一個直觀、專業的房貸試算工具。使用者可輸入房屋總價、自備款、利率、年限及寬限期，系統將自動計算每月應繳金額及總利息支出。
# 執行說明
1. 在房屋總價欄位輸入數值（單位：萬元）。
2. 設定自備款比例與貸款利率（支援百分比輸入）。
3. 選擇貸款年限並設定寬限期。
4. 點擊「計算」後，系統會自動跳轉至「結果」分頁顯示詳細報表。
# 核心功能與亮點
1. **分頁式設計**：使用 `TabControl` 整合「輸入」與「結果」頁面，避免彈跳視窗干擾操作流程。
2. 具備預設值（20% 自備款、2.15% 利率）引導使用者。
3. 點擊「計算」後自動切換至結果分頁，動線流暢。
# 計算公式
本程式採用以下公式計算每月應繳金額：

$$M = P \times \frac{r(1+r)^n}{(1+r)^n - 1}$$

* $M$: 每月應繳金額
* $P$: 貸款本金 (總價 - 自備款)
* $r$: 月利率 (年利率 ÷ 12 ÷ 100)
* $n$: 還款期數 ((貸款年限 - 寬限期) × 12)
